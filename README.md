# 🌾 田园小农场

一款清新田园风格的挂机种菜放置小游戏！

## 🎮 游戏特色

- **🌱 种植系统**: 多种农作物可供选择，不同成熟时长和收益
- **⏰ 离线挂机**: 退出页面依旧正常计时，支持离线收益
- **💰 交易系统**: 收获产物一键售卖兑换金币
- **🏪 商店系统**: 购买高级种子、解锁土地、购买加速道具
- 💾 **数据存档**: 自动保存，刷新页面游戏数据不丢失
- 📱 **移动适配**: 竖屏布局，完美适配手机端

## 🎯 玩法说明

1. 选择空地点击播种
2. 等待作物成熟
3. 成熟后点击收割
4. 在仓库售卖获得金币
5. 使用金币购买种子或解锁新土地

## 🚀 快速开始

直接打开 `index.html` 即可开始游戏，无需后端服务器。

## 🛠️ 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (ES6+)
- LocalStorage 数据持久化
